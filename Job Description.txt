1. Artificial Intelligence (AI) Engineer – Job Description

Job Title: Artificial Intelligence Engineer

Responsibilities
Design and develop AI-powered applications and solutions.
Build and deploy machine learning and deep learning models.
Develop NLP, computer vision, and generative AI applications.
Integrate AI models with web and cloud applications.
Optimize model performance and accuracy.
Work with large datasets for training and evaluation.
Collaborate with software developers and data scientists.
Monitor and maintain AI systems in production.
Required Skills
Python
TensorFlow or PyTorch
LangChain & LangGraph
OpenAI API / LLMs
NLP & Computer Vision
SQL
Git & Docker
AWS, Azure, or GCP

Job Title: Data Scientist

Responsibilities
Collect, clean, and analyze structured and unstructured data.
Build predictive and statistical models.
Perform exploratory data analysis (EDA).
Develop dashboards and reports.
Create data visualizations.
Communicate insights to stakeholders.

Improve business decisions using analytics.
Required Skills
Python
SQL
Pandas
NumPy
Scikit-learn
Power BI or Tableau
Statistics
Machine Learning
Data Visualization
